    AI Vyhledávač

    Tři módy: hledat kontakty, objevovat nové zdroje, prohloubit info o existujícím kontaktu. Aplikace paralelně zavolá Claude i ChatGPT s web search, deduplikuje napříč modely + proti tvé DB, ukáže návrhy s badge spolehlivosti.

    Najde reálné kontakty pro vybraný segment přes web search a Tvé zdroje. Cílí na decision-makery: Partner, Director, CEO.
    Vyšší max_uses web search (25 vs. 8) + vyšší max_tokens (16k vs. 8k) + agresivnější prompt. ~3–4× delší čas (2–4 min) a ~3–4× vyšší cena u API. V manuálním módu zdarma, ale v chatu Claude/ChatGPT může trvat 5–15 min.
    AI cílí přednostně na osoby s veřejně dohledatelným LinkedIn profilem (chceš jim posílat pozvánky/zprávy). Kontakty bez LinkedIn URL vyřadí. Užitečné, když plánuješ LinkedIn outreach campaign.
    Volá Claude + ChatGPT API paralelně s web search. Gemini je dostupný pouze v manuálním módu.
    1. Vygeneruj prompt
    Aplikace složí kompletní prompt vč. zdrojů, instrukcí a JSON schématu. Pak otevři Claude.ai nebo ChatGPT.com a vlož ho.
    2. Zkopíruj a vlož do chatu
    3. Vlož odpovědi zpět
    Když AI dokončí (může trvat dlouho), zkopíruj celý code block (```json ... ```) z chatu a vlož sem. Můžeš vložit jednu, dvě, nebo všechny tři odpovědi — čím víc AI dohromady, tím spolehlivější výsledek (kontakty, které najdou všechny 3 AI, jsou s nejvyšší pravděpodobností reálné).
    Odpověď z Claude (volitelně) prázdné
    Odpověď z ChatGPT (volitelně) prázdné
    Odpověď z Gemini (volitelně) prázdné
    4. Sloučit odpovědi do unikátních návrhů
    Aplikace porovná všechny odpovědi mezi sebou (odstraní duplicity napříč AI), porovná je s Tvou existující databází (odstraní kontakty, které už máš) a ukáže Ti jen unikátní nové návrhy. Pak teprve vybereš, které chceš zapsat do databáze.

    Struktura Sheetu (migrace schematu)

    Když aplikace přidá nový sloupec (např. nedávno linkedinStav), automatická migrace by ho měla doplnit do hlavičky Sheetu při prvním sync. Pokud z nějakého důvodu neproběhne (rate-limit, expired token, cached JS), klikni Migrovat — aplikace zkontroluje všechny 4 listy (Contacts, Sources, Segments, Activity) a doplní chybějící sloupce.
