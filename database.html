    AI Vyhledávač kontaktů

    V další fázi (F2): vyber segment a aplikace přes Anthropic + OpenAI API najde kontakty ze zdrojů. Výsledky z obou modelů se merge-ují, duplicity se odstraní, ke každému kontaktu badge spolehlivosti (Claude+ChatGPT / jen jeden).

    Zatím vypnuto. Nejdřív stavíme F1 — manuální správa kontaktů, import, filtry, GDrive sync. Vyhledávač přidám hned po schválení F1.
    Co bude umět:
    • Vybrat segment + případně zúžit (region, velikost firmy)
    • Paralelně zavolat Claude a ChatGPT s identickým promptem
    • Fuzzy deduplikace (IČO / email / jméno+firma / LinkedIn URL)
    • Cross-check proti existující DB — „už existuje, aktualizovat?"
    • Manuální schválení před uložením
    • Audit log — kdy a odkud kontakt přišel
